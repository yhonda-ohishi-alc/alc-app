# 前回セッションから作業を自動再開

`.claude/handoff.md` を読み込み、**提案ではなく即座に作業を実行する**。

## 手順

1. **引き継ぎファイルの読み込み**: `.claude/handoff.md` を Read ツールで読み込む
   - ファイルが存在しない場合は「引き継ぎファイルがありません。/next-session で保存してから使用してください」と案内して終了
2. **git 状態の確認**: `git status` で現在の状態を確認
3. **引き継ぎ内容を簡潔に報告** (3行以内): 何を再開するかだけ伝える
4. **「次にやること」の先頭タスクから即座に実行開始する** — 提案や確認は不要、すぐ手を動かす
   - $ARGUMENTS が指定されている場合はそちらを優先して実行
5. タスクが完了したら次のタスクに進む
6. **全タスク完了後、`plan/implementation-plan.md` を更新する**:
   - 完了したタスクのチェックボックスを `[x]` に変更
   - Phase 見出しに ✅ を追記
   - 進捗サマリーテーブルの状態を更新
   - 更新内容をコミットに含める
