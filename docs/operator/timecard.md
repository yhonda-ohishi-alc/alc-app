# タイムカード打刻

## 概要

NFC カードを使って出退勤の打刻を行います。

!!! info "開発中"
    タイムカードタブの画面は現在開発中です。

## NFC カードで打刻

1. 「タイムカード」タブをタップします
2. NFC カードをタッチします
3. カードに紐づけられた乗務員名が表示され、打刻が記録されます
4. 当日の打刻一覧が5秒間表示されます

## 打刻結果の確認

打刻が成功すると、乗務員名と打刻時刻が表示されます。当日の打刻履歴も合わせて確認できます。

## カードが読み取れない場合

- NFC カードを端末の NFC 読み取り位置にしっかりタッチしてください
- 端末の NFC 機能が有効になっているか確認してください
- カードが破損している場合は管理者に連絡してください
