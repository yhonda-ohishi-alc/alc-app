# 自動点呼

## 概要

自動点呼は、スケジュールに基づいて乗務前・乗務後の点呼を自動で実施する機能です。顔認証による本人確認から日常点検まで、一連のフローをガイドに従って進めます。

!!! info "開発中"
    自動点呼タブの画面は現在開発中です。

## 点呼の流れ

自動点呼は以下の9ステップで構成されます。

### 1. スケジュール選択

未消化の点呼スケジュールから実施するものを選択します。

### 2. 本人確認（顔認証）

カメラで顔認証を行い、本人であることを確認します。詳しくは [顔認証](face-auth.md) を参照してください。

### 3. アルコール測定

FC-1200 でアルコール濃度を測定します。詳しくは [アルコール測定](alcohol-measurement.md) を参照してください。

### 4. バイタル測定

BLE 医療機器または手動入力で体温・血圧を測定します。

![体温・血圧測定画面](../images/operator-medical.png)

- **BLE機器タブ**: 体温計・血圧計が自動検出され、測定値が表示されます
- **手動入力タブ**: 機器が接続できない場合、手動で数値を入力できます

![体温測定済み](../images/operator-medical-measured.png)

測定が完了したら「次へ」をタップします。「スキップ」で測定をスキップすることもできます。

### 5. 自己申告

健康状態の自己申告を行います。詳しくは [自己申告](self-declaration.md) を参照してください。

### 6. 安全運転判定

バイタル測定の結果と自己申告の内容から、安全運転の可否が自動で判定されます。健康基準値を超過した場合や自己申告で「はい」と回答した場合、注意喚起が表示されます。

### 7. 日常点検

車両の日常点検を実施します。詳しくは [日常点検](daily-inspection.md) を参照してください。

### 8. 指示事項確認

運行管理者からの指示事項を確認し、「確認しました」をタップします。

### 9. 完了

すべてのステップが完了すると、点呼結果のサマリーが表示されます。

## 点呼が中断された場合

点呼の途中で「最初からやり直す」をタップするか、アプリを閉じた場合、点呼は中断状態になります。中断された点呼は、再度同じスケジュールを選択することで再開できます。

## よくある質問

**Q: スケジュールが表示されない場合は？**
A: 管理者がスケジュールを登録していない可能性があります。管理者に確認してください。

**Q: 途中でやり直したい場合は？**
A: 画面左下の「最初からやり直す」をタップすると、最初のステップに戻ります。
