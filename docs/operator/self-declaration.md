# 自己申告

## 概要

自己申告は、点呼フロー中に乗務員が自身の健康状態を申告するステップです。申告内容は安全運転判定に反映されます。

## 申告項目

![自己申告画面](../images/operator-self-declaration.png)

以下の3項目について「はい」または「いいえ」で回答します。

### 疾病

体調不良や疾病の自覚症状がありますか？

### 疲労

過度な疲労を感じていますか？

### 睡眠不足

睡眠不足の自覚がありますか？

## 申告手順

1. 各項目について「はい」または「いいえ」を選択します
2. すべての項目に回答したら「送信」をタップします
3. 自動的に次のステップに進みます

## 申告内容と安全運転判定

いずれかの項目で「はい」と回答した場合、安全運転判定で注意喚起が表示されます。運行管理者にも通知され、運行の可否について判断が行われます。
