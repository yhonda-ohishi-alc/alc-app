# デバイス登録

## 概要

Google アカウントを使わずに端末を登録する方法です。管理者が発行した QR コードまたは URL を使って登録します。

## QRコードで登録

1. 管理者が表示する QR コードをカメラでスキャンします
2. 端末のデバイス名と電話番号を入力します
3. 登録が完了すると、自動的にアプリが有効化されます

## URLリンクで登録

1. 管理者から共有された URL（LINE 等）をタップします
2. デバイス登録ページが開きます
3. デバイス名と電話番号を入力し、「登録」をタップします
4. 登録が完了すると、自動的にアプリが有効化されます

## 登録コードで登録

1. 管理者から伝えられた登録コードを入力します
2. デバイス名と電話番号を入力します
3. 管理者の承認を待ちます（QR 永久フローの場合）

## 登録後の確認

登録が完了すると、端末にテナント ID とデバイス ID が保存され、以降は自動的にログインされます。
