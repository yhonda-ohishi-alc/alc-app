# アルコール測定

## 概要

FC-1200 アルコールチェッカーを使用して呼気中のアルコール濃度を測定します。

## 測定機器（FC-1200）の接続

FC-1200 は USB ケーブルで端末に接続し、WebSocket ブリッジ経由で通信します。接続状態はメンテナンス画面で確認できます。

## 測定手順

1. 点呼フロー中に「アルコール測定」ステップに進むと、自動で FC-1200 との接続を待ちます
2. 「息を吹きかけてください」と表示されたら、FC-1200 のセンサー部に向かって約5秒間息を吹きかけます

![アルコール測定画面](../images/operator-alc-measurement.png)

3. 測定が完了すると、結果が自動で記録され次のステップに進みます

## 測定結果の見かた

測定完了後、結果画面にアルコール濃度が表示されます。

![測定結果画面](../images/operator-result.png)

| 項目 | 内容 |
|------|------|
| 乗務員 | 測定対象の乗務員名 |
| アルコール濃度 | 測定値（mg/L） |
| 使用回数 | FC-1200 の累計使用回数 |
| 測定日時 | 測定を実施した日時 |
| 体温 | BLE 体温計で測定した体温 |

**0.00 mg/L** であれば「正常」と判定されます。

## 測定に失敗する場合

- FC-1200 が「未接続」の場合、メンテナンス画面で接続状態を確認してください
- 息を吹きかける時間が短すぎると測定できません（約5秒間）
- 測定直後に再測定する場合、センサーのリセットを待ってください

## よくある質問

**Q: 測定値が 0.00 以外だった場合は？**
A: 飲酒していない場合でも、口腔内のアルコール成分（マウスウォッシュ等）で反応することがあります。水で口をすすいでから 15 分以上待って再測定してください。
