# 遠隔点呼（運行者側）

## 概要

遠隔点呼は、運行管理者とビデオ通話をしながら点呼を実施する機能です。管理者がリアルタイムで映像を確認しながら、各ステップを進めます。

## 遠隔点呼の開始

「遠隔点呼」タブをタップすると、遠隔点呼モードに入ります。

![遠隔点呼 待機画面](../images/operator-remote-tenko.png)

NFC カードまたは手動 ID 入力で乗務員を識別すると、管理者からの接続を待機します。

## 着信と応答

管理者が点呼を開始すると、着信画面が表示されます。

![遠隔点呼 着信画面](../images/operator-remote-tenko-incoming.png)

- **応答**: 緑のボタンをタップするとビデオ通話が開始されます
- **拒否**: 赤のボタンをタップすると着信を拒否します

## ビデオ通話中の操作

ビデオ通話が開始されると、画面左上にカメラ映像のプレビューが表示されます。管理者がリモートで各ステップを確認します。

## 点呼の流れ

遠隔点呼のフローは自動点呼と同じ9ステップです。各ステップの画面は通常の点呼と同様ですが、左上にビデオ通話のウィンドウが常時表示されます。

### バイタル測定

![遠隔点呼 体温・血圧](../images/operator-remote-tenko-medical.png)

BLE 医療機器または手動入力で測定します。

### アルコール測定

![遠隔点呼 アルコール測定](../images/operator-remote-tenko-alc.png)

FC-1200 の接続を待ち、息を吹きかけて測定します。

### 指示事項確認

![遠隔点呼 指示確認](../images/operator-remote-tenko-instruction.png)

運行管理者からの指示事項を確認し、「確認しました」をタップします。

### 完了

![遠隔点呼 完了](../images/operator-remote-tenko-complete.png)

点呼が完了すると、結果サマリーが表示されます。「次の点呼へ」をタップして終了します。

## 通信が切れた場合

ネットワーク接続が切れた場合、自動で再接続を試みます。再接続できない場合は、通常の自動点呼に切り替えて点呼を完了させてください。

## よくある質問

**Q: ビデオ通話の映像が映らない場合は？**
A: カメラの権限が許可されているか確認してください。アプリの設定からカメラ権限を有効にしてください。

**Q: 音声が聞こえない場合は？**
A: 端末の音量を確認してください。また、マイクの権限が許可されているか確認してください。
