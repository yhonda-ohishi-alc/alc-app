# はじめに

## 概要

ALC アプリは、業務用アルコール検知・点呼管理システムです。乗務員の本人確認（顔認証）、アルコール測定、バイタル測定、日常点検などを一連の流れで実施できます。

## 端末のセットアップ

1. Android タブレットに ALC アプリをインストールします
2. アプリを起動し、ログインまたはデバイス登録を行います
3. メンテナンス画面で FC-1200（アルコールチェッカー）と BLE 医療機器の接続を確認します

## ログイン方法

### Google アカウントでログイン

管理者から付与された Google アカウントでログインします。

### デバイス登録でログイン

管理者が発行した QR コードまたは URL を使ってデバイスを登録します。詳しくは [デバイス登録](device-registration.md) を参照してください。

## 画面の見かた

アプリのメイン画面は、上部のタブで機能を切り替えます。

![メイン画面（通常点呼）](../images/operator-normal-nfc.png)

| タブ | 機能 |
|------|------|
| 通常点呼 | NFC カードで乗務員を識別し、アルコール測定・バイタル測定を実施 |
| 自動点呼 | スケジュールに基づく点呼フロー（顔認証→測定→自己申告→日常点検→指示確認） |
| 遠隔点呼 | 運行管理者とビデオ通話しながら点呼を実施 |
| タイムカード | NFC カードで出退勤の打刻 |

画面左側にはフローのステップが表示され、現在のステップがハイライトされます。画面下部の「顔登録」「メンテナンス」リンクから、それぞれの設定画面に移動できます。

### メンテナンス画面

メンテナンス画面では、測定機器の接続状態を確認できます。

![メンテナンス画面](../images/operator-maintenance.png)

- **FC-1200 アルコールチェッカー**: USB 接続 → WebSocket ブリッジで通信
- **BLE 医療機器（体温計・血圧計）**: Bluetooth 接続 → WebSocket ブリッジで通信

![BLE 接続中](../images/operator-maintenance-ble.png)

BLE ブリッジが接続されると、検出された機器が表示されます。
