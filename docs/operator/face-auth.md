# 顔認証

## 概要

顔認証は、点呼時の本人確認に使用します。事前に顔を登録し、点呼実施時にカメラで本人であることを確認します。

## 顔の登録

### 登録手順

1. メイン画面下部の「顔登録」リンクをタップします
2. カメラが起動し、顔の検出が始まります
3. 画面の枠内に顔が収まるように位置を調整します
4. 検出条件（顔サイズ、正面向き、まばたき）がすべて緑色になったら、確認ダイアログが表示されます
5. 「登録する」をタップして登録を完了します

![顔登録確認画面](../images/operator-face-register.png)

### 撮影のコツ

- 正面を向き、カメラから 30〜50cm の距離を保ちます
- 明るい場所で撮影してください（逆光を避ける）
- メガネ・マスクは外してください
- 画面下部のステータスを確認し、すべて緑色（OK）になるのを待ちます

## 顔認証の実施

点呼フロー中に自動で顔認証ステップに進みます。

![顔認証画面](../images/operator-face-auth.png)

画面下部に検出ステータスが表示されます:

| 項目 | 内容 |
|------|------|
| 顔検出 | 顔が検出されたか（数値は検出数） |
| 顔サイズ | 顔が十分な大きさか |
| まばたき | まばたきが検出されたか（なりすまし防止） |
| 信頼度 | 認証の信頼度（100% で最高） |
| 正面向き | 正面を向いているか |
| 特徴量 | 特徴量が正常に抽出されたか |

すべての項目が緑色のチェックマークになると、自動的に認証が完了し次のステップに進みます。

## 認証に失敗する場合

- 照明が暗い場合は明るい場所に移動してください
- カメラに汚れがないか確認してください
- メガネ・マスクを外してください
- 登録時と大きく外見が変わった場合は再登録が必要です

## 再登録が必要な場合

管理者が乗務員の顔登録をリセットした場合や、モデルバージョンが更新された場合に再登録が求められます。メイン画面下部の「顔登録」から再度登録してください。
