# 日常点検

## 概要

日常点検は、点呼フロー中に車両の安全状態を確認するステップです。8つの点検項目について OK/NG を記録します。

## 点検項目

![日常点検画面](../images/operator-daily-inspection.png)

以下の8項目を点検します。

| 項目 | 確認内容 |
|------|----------|
| ブレーキ | ブレーキの効き具合 |
| タイヤ | タイヤの空気圧・摩耗・損傷 |
| 灯火類 | ヘッドライト・ウインカー・ブレーキランプ等 |
| ハンドル | ハンドルの操作感・遊び |
| ワイパー | ワイパーの動作・ゴムの状態 |
| ミラー | バックミラー・サイドミラーの状態 |
| 警音器 | クラクションの動作 |
| シートベルト | シートベルトの状態 |

## 点検の実施手順

1. 各項目について「OK」または「NG」をタップします
2. すべての項目を点検したら「送信」をタップします

## NG項目がある場合

NG 項目がある場合、点呼はキャンセルされます。車両の整備・修理を行い、問題が解消されてから再度点呼を実施してください。
